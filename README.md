# PRODIGY_GA_02 - Image Generation with Pre-trained Models

## Objective
Generate images from text prompts using pre-trained generative AI models.

## Description
This project uses Stable Diffusion, a pre-trained image generation model, to create images from textual descriptions. The model converts text prompts into realistic and creative images.

## Technologies Used
- Python
- Google Colab
- Stable Diffusion
- Diffusers Library

## Features
- Generates images from text prompts
- Uses a pre-trained Stable Diffusion model
- Demonstrates Generative AI in image synthesis

## Sample Prompt
"A futuristic city at sunset with flying cars"
